Пользователи (Users)
-------------------------
ID (уникальный идентификатор)
Имя пользователя (Username)
Пароль (Password)
Email
Дата регистрации (RegistrationDate)


Категории товаров (Categories)
-------------------------
ID
Название категории (CategoryName)
Описание (Description)


Товары (Products)
-------------------------
ID
Название товара (ProductName)
Описание (Description)
Цена (Price)
ID категории (CategoryID) - внешний ключ к таблице Categories
Признак новинки (IsNew)



Резервации (Reservations)
-------------------------
ID
ID пользователя (UserID) - внешний ключ к таблице Users
ID товара (ProductID) - внешний ключ к таблице Products
Дата резервации (ReservationDate)
Статус (Status) - например, "в обработке", "подтверждено", "отменено"
