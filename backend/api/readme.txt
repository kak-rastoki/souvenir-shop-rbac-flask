
Модуль API (api)
--------------------------------------------------
Описание:
    Этот модуль предоставляет API-маршруты для доступа к данным веб-приложения.
    В настоящее время он включает в себя маршруты для получения информации
    о товарах и аватарах пользователей.

Импортируемые модули и их назначение:
    - `io`: Используется для работы с потоками ввода-вывода, в частности,
      для отправки изображений в виде байтовых данных.
        *Где используется:* `product_image`, `get_avatar`.

    - `flask`: Веб-фреймворк для Python. Используются следующие компоненты:
        - `send_file`: Отправляет файл (в данном случае, изображение)
          в ответ на запрос.
            *Где используется:* `product_image`, `get_avatar`.
        - `render_template`: (Не используется в данном модуле) Рендерит
          HTML-шаблон.
        - `redirect`: Перенаправляет пользователя на другой URL.
            *Где используется:* `get_avatar`.
        - `url_for`: Генерирует URL-адрес для указанной функции просмотра.
            *Где используется:* `get_avatar`.
    - `models`: (models.py) Содержит определения моделей базы данных.
        - `Product`: Модель для представления информации о товаре.
            *Где используется:* `productCard`, `product_image`.
        - `Users`: Модель для представления информации о пользователе.
            *Где используется:* `get_avatar`.
    - `api`: (.)
        - `api_bp`: Экземпляр Blueprint для этого модуля.
