Модуль авторизации и регистрации (auth)
--------------------------------------------------
Описание:
    Этот модуль содержит функциональность аутентификации и регистрации
    пользователей веб-приложения. Он предоставляет маршруты для отображения
    страницы регистрации/входа, регистрации новых пользователей, входа
    существующих пользователей и выхода из системы.

Ключевые особенности:
    - Регистрация новых пользователей с проверкой введенных данных.
    - Аутентификация существующих пользователей с проверкой логина и пароля.
    - Поддержка сессий для хранения информации о вошедшем пользователе.
    - Использование Flask-Login для управления аутентификацией пользователей.
    - Хеширование паролей для безопасного хранения в базе данных.

Структура:
    - `auth_bp`: Экземпляр Blueprint, определяющий маршруты и ресурсы
      модуля аутентификации.
    - `models.py`: Содержит определения моделей базы данных, используемых
      для представления данных о пользователях.

Импортируемые модули и их назначение:
    - `flask_login`: (Импортируются отдельные компоненты) Расширение Flask для
      управления аутентификацией пользователей.
        - `login_user`: Выполняет вход пользователя в систему.
            *Где используется:* В функции `login`.
        - `logout_user`: Выполняет выход пользователя из системы.
            *Где используется:* В функции `logout`.
        - `login_required`: Декоратор, требующий аутентификации пользователя
          для доступа к маршруту.
            *Где используется:* В функции `logout`.
    - `werkzeug.security`: (Импортируются отдельные компоненты) Библиотека
      для работы с паролями.
        - `generate_password_hash`: Генерирует хеш пароля для безопасного
          хранения в базе данных.
            *Где используется:* В функции `registration`.
        - `check_password_hash`: Проверяет, соответствует ли введенный пароль
          хешу, хранящемуся в базе данных.
            *Где используется:* В функции `login`.
    - `flask`: (Импортируются отдельные компоненты) Веб-фреймворк для Python.
        - `render_template`: Рендерит HTML-шаблон и возвращает его в качестве
          ответа.
            *Где используется:* В функции `signup`.
        - `url_for`: Генерирует URL-адрес для указанной функции просмотра.
            *Где используется:* Для перенаправления пользователей на другие
              страницы.
        - `request`: предоставляет доступ к входящим данным запроса (форма, json).
            *Где используется:* для получения данных с форм
        - `redirect`: Перенаправляет пользователя на другой URL.
            *Где используется:* Для перенаправления пользователей после
              выполнения определенных действий (например, регистрации или входа).
        - `session`: Управляет данными сессии пользователя.
            *Где используется:* для очистки сессии.
        - `flash`: для создания flash сообщений
            *Где используется:* для передачи flash сообщений
    - `auth`: (.) Текущий пакет (модуль).
        - `auth_bp`: Экземпляр `Blueprint` для модуля `auth`.
            *Где используется:* Для определения маршрутов и ресурсов модуля.
    - `models`: (models.py) Модуль, содержащий определения моделей базы
      данных.
        - `db`: Экземпляр `SQLAlchemy`, используемый для работы с базой данных.
            *Где используется:* Для выполнения запросов к базе данных.
        - `Users`: Модель для представления информации о пользователях.
            *Где используется:* Для выполнения запросов к таблице `Users`.



Зависимости:
    - flask: Веб-фреймворк для Python.
    - flask_login: Расширение Flask для управления аутентификацией пользователей.
    - werkzeug: Библиотека для работы с HTTP-запросами и ответами.
    - models: (models.py) Содержит определения моделей.
