
Модуль Панели администратора
--------------------------------------------------
Описание:
    Этот модуль содержит функциональность панели администратора веб-приложения.
    Он предоставляет интерфейс для управления продуктами, пользователями,
    категориями и мастерами. Доступ к функциональности панели
    администратора ограничен пользователями с ролью администратора.


Импортируемые модули и их назначение:
    - `flask_login`: (Импортируется целиком)
        - `login_required`: Декоратор, требующий аутентификации пользователя
          для доступа к маршруту.
            *Где используется:* На всех маршрутах панели администратора.
        - `current_user`: Объект, представляющий текущего аутентифицированного
          пользователя.
            *Где используется:* В декораторе `@admin_required`.
    - `datetime`: (Импортируется целиком) Модуль Python для работы с датой
       и временем.
         *Где используется:* В функции `edit_user`, для преобразования строки даты
         в объект `date`.
    - `base64`: (Импортируется целиком) Модуль Python для кодирования и
      декодирования данных в формате Base64.
        *Где используется:* В функции `get_user_api`, для кодирования аватара
          пользователя в формат Base64.


    - `decorators`: (decorators.py) Модуль, содержащий декоратор
      `@admin_required`.
        - `admin_required`: Декоратор, ограничивающий доступ к маршрутам
          только пользователям с ролью "admin".
            *Где используется:* На всех маршрутах панели администратора.
    - `models`: (models.py) Модуль, содержащий определения моделей базы
      данных.
        - `db`: Экземпляр `SQLAlchemy`, используемый для работы с базой данных.
            *Где используется:* Для выполнения запросов к базе данных.
        - `Users`: Модель для представления информации о пользователях.
            *Где используется:* Для выполнения запросов к таблице `Users`.
        - `Masters`: Модель для представления информации о мастерах.
            *Где используется:* Для выполнения запросов к таблице `Masters`.
        - `Categories`: Модель для представления информации о категориях.
            *Где используется:* Для выполнения запросов к таблице `Categories`.
        - `Product`: Модель для представления информации о продуктах.
            *Где используется:* Для выполнения запросов к таблице `Product`.
        - `Genders`: Модель для представления информации о поле (гендере).
            *Где используется:* Для выполнения запросов к таблице `Genders`.

    - `admin`: (.) Текущий пакет (модуль).
        - `admin_bp`: Экземпляр `Blueprint` для модуля `admin`.
            *Где используется:* Для определения маршрутов и ресурсов модуля.
